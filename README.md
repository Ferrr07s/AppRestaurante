 # 🍽️ AppRestaurante

API REST desarrollada con **Spring Boot** para la gestión integral de un restaurante.

## 📋 Descripción

Sistema backend que permite administrar las operaciones principales de un restaurante,
incluyendo la gestión de clientes, mesas, platos, pedidos y reservas.

## 🛠️ Tecnologías

- Java
- Spring Boot
- Spring Data JPA
- Maven

## 📦 Módulos principales

- **Clientes** – Registro y gestión de clientes
- **Mesas** – Control de disponibilidad de mesas
- **Platos** – Catálogo de productos del menú
- **Pedidos** – Creación y seguimiento de pedidos
- **Reservas** – Gestión de reservas de mesas
